# Binary Search

Binary Search is a **divide-and-conquer algorithm** used to find an element in a **sorted array** efficiently.

---

## 🚀 How it works

1. Start with the middle element
2. If it matches the target → done
3. If target is smaller → search left half
4. If target is larger → search right half
5. Repeat until found or range becomes empty

---


## ⚠️ Requirement

- Array must be **sorted**

---

## 📌 Example

Array: `[1, 3, 5, 7, 9]`  
Target: `7` → Output index: `3`

---

## 💡 Why it matters

Binary Search is one of the most important algorithms in computer science because it drastically reduces search time compared to linear search.
