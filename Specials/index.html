VERY IMPORTANT
GCD (Euclid Algorithm)
দুইটি সংখ্যার মধ্যে সবচেয়ে বড় যে সংখ্যা দিয়ে উভয়কে ভাগ করা যায় সেটাই GCD। Euclid Algorithm ব্যবহার করলে দ্রুত GCD বের করা যায়। 
#include <stdio.h>

int gcd(int a, int b){
    if(b == 0) return a;
    return gcd(b, a % b);
}

int main(){
    printf("%d", gcd(24, 36));
}
IMPORTANT
LCM (Least Common Multiple)
দুইটি সংখ্যার সবচেয়ে ছোট সাধারণ গুণিতককে LCM বলে। সূত্র: LCM = (a × b) / GCD 
#include <stdio.h>

int gcd(int a, int b){
    if(b == 0) return a;
    return gcd(b, a % b);
}

int lcm(int a, int b){
    return (a / gcd(a,b)) * b;
}

int main(){
    printf("%d", lcm(12, 18));
}
EXAM FAVORITE
Sum of Dhara (Arithmetic Series)
ধারা মানে একটি নির্দিষ্ট প্যাটার্নে সংখ্যা যোগ করা। Arithmetic series এর সূত্র: S = n/2 × (2a + (n-1)d) 
#include <stdio.h>

int main(){
    int a = 1, d = 2, n = 5;
    int sum = (n * (2*a + (n-1)*d)) / 2;
    printf("%d", sum);
}
COMMON
Factorial (n!)
কোনো সংখ্যার factorial মানে ১ থেকে সেই সংখ্যার পর্যন্ত সব গুণফল। যেমন: 5! = 5×4×3×2×1 
#include <stdio.h>

int main(){
    int n = 5, fact = 1;

    for(int i = 1; i <= n; i++){
        fact *= i;
    }

    printf("%d", fact);
}
